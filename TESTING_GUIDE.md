
# 🧪 VideoMaster AI: Testing Guide (Aasaan Guide)

Aapka app sahi se kaam kar raha hai ya nahi, ise check karne ke liye niche diye gaye steps follow karein:

### 1. System Health Check (Sabse Zaroori)
- App ke **Dashboard** par jayein.
- Wahan **"Run System Diagnostics"** button par click karein.
- Ye page check karega ki:
  - Database juda hai? ✅
  - Login system taiyar hai? ✅
  - AI "Brain" (Gemini) connected hai? ✅

### 2. AI Script & Thumbnail Test
- `/editor` page par jayein.
- **Viral Script Writer** mein koi topic likhein (e.g., "How to earn money online").
- "Write Script" button dabayein. Agar script generate hoti hai, toh aapka AI setup perfect hai.

### 3. Ads & Reward Test
- **Dashboard** par niche scroll karein.
- **"Watch Ad & Earn Credits"** button par click karein.
- 5 second wait karein. Agar aapke credits +10 ho jaate hain, toh aapka "Paisa Kamane ka System" active hai.

### 4. Search Ranking Test
- Google par `site:studio-9489287013-59986.web.app` search karein.
- Shuruat mein agar kuch nahi dikhta toh 24 ghante wait karein, Google scan kar raha hoga.

Aapka app ab testing ke liye puri tarah taiyar hai! 🚀
