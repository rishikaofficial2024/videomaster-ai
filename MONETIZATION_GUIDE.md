
# 💰 VideoMaster AI: Ads se Paisa Kamane ka Full Plan

Aapka app ab ek asli "Ad-Revenue Engine" ban chuka hai. Maine sabhi modern ad formats ismein integrate kar diye hain taaki aap har user se paise kama sakein.

### 1. Verification Checklist (Sabse Zaroori) ✅
1. **app-ads.txt**: Maine `public/app-ads.txt` add kar di hai. Google AdSense ise check karega taaki wo samajh sake aap hi asali malik hain.
2. **Domain Status**: [Google AdSense](https://adsense.google.com) mein jayein aur apna domain `studio-9489287013-59986.web.app` add karein.
3. **Review Time**: Google usually 2 se 14 din leta hai site approve karne mein. Jab site green ho jayegi, ads apne aap dikhne lagenge.

### 2. Rewarded Video Ads (Highest Earning) ✅
- **Kahan Hai?**: Dashboard par "Watch Ad & Get Credits" section.
- **Kaise Kaam Karta Hai?**: User 15 second ka video ad dekhega aur use +20 AI Credits milenge.
- **Fayda**: In ads ka CPM (paisa) normal banners se 5-10 guna zyada hota hai. Maine iska coding logic `src/app/dashboard/page.tsx` mein set kar diya hai.

### 3. Subscription Model (Pro Plan ₹99) ✅
- **Kahan Hai?**: `/premium` page.
- **Kaise Kaam Karta Hai?**: User jab ₹99 ka plan lega, toh wo ad-free experience aur unlimited credits payega.
- **Status**: Maine iska Firestore logic set kar diya hai. Jaise hi user pay karega (simulated), uska profile "isPremium: true" ho jayega.

### 4. Smart Display Banners ✅
- **Kahan Hai?**: Home page, Dashboard aur Templates ke niche.
- **Publisher ID**: Aapki ID `pub-8946933317699938` har jagah set hai.
