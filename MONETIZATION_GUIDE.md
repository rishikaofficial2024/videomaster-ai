
# 💰 VideoMaster AI: Paisa Kamane Ka Plan (Aasaan Guide)

Aapka app ab ek "Money-Making Machine" banne ke liye taiyar hai. Niche padhein ki ads aur subscriptions se paise aapke bank mein kaise aayenge:

### 1. Google AdSense (Website Par Ads) ✅
AdSense se paise kamane ke liye ye steps follow karein:
1. [Google AdSense](https://adsense.google.com) par account banayein.
2. Left menu mein **Account > Settings > Account Information** par jayein.
3. Wahan se apni **Publisher ID** (jo `pub-` se shuru hoti hai) copy karein.
4. App ki file `src/app/layout.tsx` kholiye aur jahan `ca-pub-PLACEHOLDER` likha hai, wahan apni ID paste kar dein (e.g. `ca-pub-123456789`).
5. Jab log aapki site par aayenge aur ads dikhenge, Google har mahine ki 21 tareek ko aapke bank mein paise bhej dega.

### 2. Google AdMob (Mobile App Par Ads) ✅
Agar aap app ko Play Store par dalte hain:
1. [Google AdMob](https://admob.google.com) par jayein aur "App" create karein.
2. Wahan se **App ID** aur **Ad Unit ID** lein.
3. Dashboard par jo "Watch Ad" button hai, wahan ye IDs kaam aayengi. User 5 second ka video ad dekhega aur aapki kamai hogi.

### 3. Subscriptions (Pro Plans) ✅
- **Free Users**: Inhe sirf 100 credits milte hain.
- **Pro Users (₹99)**: Inhe unlimited access milta hai.
- **Paisa Kaise Milega?**: 
  1. [Razorpay](https://razorpay.com) ya **Instamojo** par account banayein.
  2. Apna "Payment Link" banayein aur usey `premium/page.tsx` mein upgrade button par laga dein.
  3. User payment karega aur paisa turant aapke bank mein aayega.

### 4. Reward Ads (Free Credits for Users)
Maine Dashboard par ek system set kiya hai:
- User ko free credits chahiye? Wo "Watch Ad" button dabayega.
- User 5-10 second ka ad dekhega.
- Aapko ad ka paisa milega, aur user ko +10 credits. Dono ka fayda!

---

**Maine kya-kya set kar diya hai?**
- ✅ **Ad Slots**: Pure app mein ads dikhane ki jagah (Banners) bani hui hai.
- ✅ **AdSense Script**: Layout mein pehle se hi script ready hai, bas ID daalni hai.
- ✅ **Pricing Section**: Landing page par professional plans dikh rahe hain.

Aapka app ab **READY-TO-EARN** hai! 🚀💰
