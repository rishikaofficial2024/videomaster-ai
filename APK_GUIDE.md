
# 📱 APK Kaise Banayein? (Final Steps)

GitHub par code push karte hi APK apne aap banne lagega. Niche diye gaye commands ko Terminal mein dalein:

### 1. Code Ko GitHub Par Dalein (Push)
Studio ke neeche **Terminal** kholein aur ye commands ek-ek karke copy-paste karein:

```bash
git init
git remote add origin https://github.com/rishikaofficial2024/videomaster-ai.git
git add .
git commit -m "Initial Studio Build"
git branch -M main
git push -u origin main
```

### 2. APK Bante Huye Dekhein
👉 **Direct Link**: [GitHub Actions Status](https://github.com/rishikaofficial2024/videomaster-ai/actions)
- Jab **Green Tick (✅)** aa jaye, toh uspar click karke niche `VideoMasterAI-Final-APK` download karein.

Aapka app ab poori tarah business ke liye taiyar hai! 🚀💰
