# 🚀 How to Fix "Site Not Found" (Deploy Protocol)

Bhaai, "Site Not Found" ka matlab hai ki aapka code abhi tak Google ke Cloud par upload nahi hua hai. Ise 2 minute mein fix karein:

### Step 1: Open Terminal
Studio mein sabse upar Left side mein `≡` (Menu) par click karein > `Terminal` > `New Terminal`.

### Step 2: Run Deploy Command
Terminal mein ye command niche se copy karke paste karein aur **Enter** dabayein:
```bash
npm run web:deploy
```

### Step 3: Wait for Success
Jab Terminal mein **"✔  Deploy complete!"** likha hua aa jaye, toh samajh lijiye aapka app poori duniya ke liye live ho gaya hai.
