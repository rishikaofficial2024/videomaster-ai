## 🛡️ Security & Setup
1. **SHA-256 Fingerprint**: Build logs (GitHub Actions) mein "Extract SHA-256" step dekhein. Usey Firebase Project Settings mein "Add Fingerprint" mein paste karein.
2. **Authorized Domains**: Browser URL ko Firebase Console > Authentication > Settings mein add karein (auth/unauthorized-domain fix).
3. **SMS Policy**: Firebase Console mein 'SMS Region Policy' par jaakar **India (IN)** ko 'Allow' karein.
