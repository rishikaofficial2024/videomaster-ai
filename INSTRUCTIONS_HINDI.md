# 🗝️ VideoMaster AI: Terminal Command aur APK Download (Hindi)

Bhaai, Terminal dhoondna aur APK download karna ab ekdum aasaan hai, chahe aap **Mobile** use kar rahe hon ya Laptop. Bas ye steps follow karein:

### ⚠️ Zaroori Warning (Don't Be Confused) ✅
Aapki screen par jo **"Route: Static"** ya **"Preferences"** wala chota box dikh raha hai, woh Terminal **NAHI** hai. Woh sirf app ki info hai. Terminal dhoondne ke liye niche dekhein:

### 1. Terminal Kahan Hai? (Where is the Terminal?) ✅

**Mobile Phone par (Asli Steps):**
- **Step 1**: Screen ke sabse upar baayein (Top-Left) kone mein dekhein. Wahan 3 horizontal lines (Menu ≡) hongi. Uspar click karein.
- **Step 2**: Menu ke andar niche scroll karein, wahan `Terminal` likha hoga. Uspar click karein.
- **Step 3**: Fir `New Terminal` select karein.
- **Step 4**: Screen ke niche ek **Kaala Box** khul jayega, wahi asli Terminal hai.

**Laptop/PC par:**
- **Shortcut**: Keyboard par **Ctrl** aur **~** (tilde) key ek saath dabayein.
- **Mouse**: Screen ke sabse niche panel mein dekhein, jahan `Terminal` likha hai.

### 2. Command Kaise Chalayein? ✅
- Jab asli Kaala Box (Terminal) khul jaye, toh niche wala code **Copy** karein.
- Terminal ke andar **Daba kar rakhein (Long Press)** aur **PASTE** select karein, fir keyboard ka **Enter (Go)** dabayein:
```bash
npm run mobile:push
```
