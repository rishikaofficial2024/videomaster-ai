
# 🗝️ VideoMaster AI: Terminal Command Kaise Chalayein? (Hindi)

Bhaai, Terminal dhoondna ab ekdum aasaan hai. Bas ye 3 steps follow karein:

### 1. Terminal Kahan Hai? ✅
- Apni screen ke **SABSE NICHE** dekhein. 
- Wahan ek pati (bar) hogi jismein **Terminal** likha hoga. 
- Us par click karein. Ek kaala box khulega.

### 2. Command Kya Hai? ✅
- Us kaale box (Terminal) mein ye niche wala code **Copy** karke **Paste** karein:
```bash
npm run mobile:push
```
- Paste karne ke baad apne keyboard par **ENTER** dabayein.

### 3. Hogaya! Build Check Karein ✅
- Command chalne ke baad aapka code GitHub par chala jayega. 
- APK bante huye dekhne ke liye is link par jayein:
👉 [GitHub Build Status](https://github.com/rishikaofficial2024/videomaster-ai/actions)
