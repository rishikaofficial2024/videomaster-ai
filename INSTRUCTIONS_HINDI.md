# 🗝️ VideoMaster AI: Terminal Command aur APK Download (Hindi)

Bhaai, Terminal dhoondna aur APK download karna ab ekdum aasaan hai. Maine origin error ko bhi fix kar diya hai.

### 1. Terminal Kahan Hai? ✅

**Mobile Phone par:**
- **Step 1**: Screen ke sabse upar baayein (Top-Left) kone mein **Menu ≡** par click karein.
- **Step 2**: Menu mein niche scroll karke **Terminal** aur fir **New Terminal** select karein.
- **Step 3**: Screen ke niche ek **Kaala Box** khul jayega.

### 2. Command Kaise Chalayein? ✅
Ab sirf ek hi command se aapka GitHub link bhi thik ho jayega aur code bhi push ho jayega:

```bash
npm run mobile:push
```

### 3. APK Download Kaise Karein? ✅
- Command chalne ke baad is link par jayein:
👉 [GitHub Build Status](https://github.com/rishikaofficial2024/videomaster-ai/actions)
- Jab build **Green ✅** ho jaye, toh usey click karke sabse niche **Artifacts** mein **VideoMasterAI-Final-APK** download karein.
