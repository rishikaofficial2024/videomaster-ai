# 🗝️ VideoMaster AI: Terminal Command aur APK Download (Hindi)

Bhaai, Terminal dhoondna aur APK download karna ab ekdum aasaan hai, chahe aap **Mobile** use kar rahe hon ya Laptop. Bas ye steps follow karein:

### 1. Terminal Kahan Hai? (Where is the Terminal?) ✅

**Laptop/PC par:**
- **Shortcut**: Keyboard par **Ctrl** aur **~** (tilde) key ek saath dabayein.
- **Mouse**: Screen ke sabse niche panel mein dekhein, jahan `Terminal` likha hai.

**Mobile Phone par (Khas Instructions):**
- **Swipe Up**: Apni screen ke bilkul niche edge se upar ki taraf swipe karein, wahan panel chupa hota hai.
- **Menu Option**: Upar left mein 3 lines (Menu) par click karein aur `Terminal` > `New Terminal` select karein.
- **Long Press**: Command paste karne ke liye Terminal ke andar thodi der ungli dabakar rakhein, fir `Paste` dabayein.

### 2. Command Kaise Chalayein? ✅
- Jab Terminal (kaala box) khul jaye, toh niche wala code **Copy** karein.
- Terminal ke andar **Paste** karein aur **ENTER** (Search/Go) dabayein:
```bash
npm run mobile:push
```
