# 🗝️ VideoMaster AI: Terminal Command aur APK Download (Hindi)

Bhaai, Terminal dhoondna aur APK download karna ab ekdum aasaan hai. Bas ye steps follow karein:

### 1. Command Kaise Chalayein? ✅
- Apni screen ke **SABSE NICHE** dekhein. 
- Wahan ek pati (bar) hogi jismein **Terminal** likha hoga. 
- Us kaale box mein ye niche wala code **Copy** karke **Paste** karein aur **ENTER** dabayein:
```bash
npm run mobile:push
```

### 2. APK Download Kaise Karein? ✅
- Command chalne ke baad is link par jayein:
👉 [GitHub Build Status](https://github.com/rishikaofficial2024/videomaster-ai/actions)
- Sabse upar wala build (jo **Green ✅** ho) usey click karke kholein.
- Page ke sabse niche jayein, wahan **Artifacts** mein **VideoMasterAI-Final-APK** likha hoga.
- Uspar click karein, aapka mobile app download ho jayega!
