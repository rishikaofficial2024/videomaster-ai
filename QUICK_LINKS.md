### 1. Branding & Name
- **Public Name**: `VideoMaster AI`
- **Support Email**: `rishikaofficial2024@gmail.com`
- 👉 **[Project Settings](https://console.firebase.google.com/project/studio-9489287013-59986/settings/general)**

### 2. Firebase Auth (Direct Links)
- 👉 **[Enable Sign-in Providers](https://console.firebase.google.com/project/studio-9489287013-59986/authentication/providers)** -> Google, Phone aur GitHub ko **ON** karein.
- 👉 **[Add Authorized Domain](https://console.firebase.google.com/project/studio-9489287013-59986/authentication/settings)** -> `videomaster-ai.tech` wahan add karein.

### 3. GitHub Integration (APK Build)
- **Action**: Terminal mein ye command chalayein: 
  ```bash
  npm run mobile:push
  ```
- **Build Status**: [Yahan APK check karein](https://github.com/rishikaofficial2024/videomaster-ai/actions).

### 4. Security Site Key
- **File**: `src/firebase/config.ts`
- **Field**: `appCheckSiteKey`
- 👉 **[Get Site Key Here](https://www.google.com/recaptcha/admin/create)** (Select V3)
