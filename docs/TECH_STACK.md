# 💻 VideoMaster AI: Technical Infrastructure

Your studio is built using the industry's most modern and stable software architecture.

### 1. Frontend Framework: Next.js 15 (React)
- **Why?**: It provides "Server Components" which make your app load up to 2x faster than traditional websites. 
- **Coding Style**: Modern Functional Components with Hooks (useEffect, useState).

### 2. Logic Language: TypeScript
- **Why?**: Unlike plain JavaScript, TypeScript catches errors *before* they happen. It's the standard for professional "Elite" software.

### 3. Database: Cloud Firestore
- **Why?**: A NoSQL real-time database that scales automatically. Your users' projects and credits are synced globally in milliseconds.

### 4. AI Engine: Google Genkit & Gemini
- **Why?**: Genkit is Google's newest framework for building AI agents. It integrates directly with Gemini 1.5 Pro/Flash for high-intelligence reasoning.

### 5. Styling Engine: Tailwind CSS & Radix UI
- **Why?**: Allows for a custom "VN-Style Editor" look without slowing down the browser. Uses CSS variables for the Obsidian & Azure theme.

### 6. Mobile Bridge: Capacitor
- **Why?**: It converts your high-performance web code into a native Android application. This allows your app to run on both the web and as an APK.

**Your technical node is stabilized and optimized for production scale.** 🚀🛡️💎
