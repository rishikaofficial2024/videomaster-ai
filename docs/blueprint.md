# **App Name**: VideoMaster AI

## Core Features:

- Smart Workspace Authentication: Secure user login via email and Google authentication using Firebase Auth to keep projects synchronized across all mobile devices.
- Advanced Timeline Editor: A high-performance mobile video editor interface allowing for precise trimming, cutting, merging, and cropping with real-time preview.
- Intelligent Caption Tool: An AI-powered reasoning tool that automatically generates and synchronizes captions and subtitles based on the video's audio content.
- Content Alchemist Tool: AI-driven tool that analyzes video scenes to suggest optimized hashtags, SEO-friendly titles, and creative descriptions.
- FX & Audio Studio: A comprehensive library of background music and professional video filters to enhance production quality.
- Cloud Project Management: Reliable storage of video drafts, editing states, and user data using Firebase Firestore and high-speed Cloud Storage.
- Premium Tier & Monetization: Integrated AdMob banner/interstitial ads and a premium subscription system for high-resolution exports and ad-free experience.
- Admin Management Console: A centralized dashboard for managing users, monitoring analytics, and sending push notifications.

## Style Guidelines:

- Primary color: Electric Indigo (#7C3AED), a vibrant hue that communicates high-tech creativity.
- Background color: Deep Charcoal with a hint of purple (#121017) for a cinema-grade dark mode experience.
- Accent color: Sky Glow (#738AFF), used for interactive highlights and high-contrast call-to-actions.
- Headline font: 'Space Grotesk' for a computerized, modern-tech aesthetic. Body font: 'Inter' for superior readability.
- Thin-weight, sharp geometric icons that mirror the precision of professional editing software.
- A structured, modular grid layout that prioritizes the video canvas and editing timeline.
- Fluid micro-interactions and sleek transitions between editing modes, utilizing fade effects to focus on content.
